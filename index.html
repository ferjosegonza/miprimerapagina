MIS TRABAJOS:

SIGEAR

Descripción
Participé en un equipo con el que desarrollamos un sistema web para la gestión de archivos de la Comisión por la Memoria, un área dedicada a proporcionar información a los ciudadanos interesados en conocer denuncias relacionadas con los derechos humanos. La plataforma facilita la administración de estos documentos, permitiendo la búsqueda de información. También proporciona una herramienta OCR para documentos TIFF, permitiendo realizar búsquedas en ellos.

Tecnologías
Se eligieron Slim Framework, plantillas Twig y una base de datos PostgreSQL para gestionar todo el sistema web de la Comisión por la Memoria. Slim Framework, al ser un microframework, proporciona una estructura liviana y eficiente para construir aplicaciones web. Se utiliza la plantilla Twig por su simplicidad y flexibilidad en la creación de vistas y plantillas. PostgreSQL, un sólido sistema de gestión de bases de datos relacionales, se utiliza para un almacenamiento y recuperación eficientes de datos.
